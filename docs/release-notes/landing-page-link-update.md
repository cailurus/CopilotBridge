# Landing page link update checklist

For each release, review and update links in `landing/index.html`:

- **Download DMG** button: keep this as `https://github.com/cailurus/CopilotBridge/releases/latest/download/CopilotBridge-<version>.dmg`.
- **View Releases** button: keep this as `https://github.com/cailurus/CopilotBridge/releases`.
- **GitHub** footer link: keep this as `https://github.com/cailurus/CopilotBridge`.

If you keep `latest/download`, only the filename must stay consistent.
If the repository owner/name changes, update all three links in the same release PR.
